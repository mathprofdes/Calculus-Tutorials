Empty Point Set#

Description#

This is simply a convince option for creating a point set with no points. Once created it is simply a point set, see Point Set for more details on the input/edit dialog for point sets and their options.

The reason you might create an empty point set is to use the Shift-Click adding and deleting of points on the graph using the mouse. This comes in handy when examining curve fitting in Linear Algebra. You can also create an empty point set by inserting a point set and leaving the point cells blank or by inputting [] into the CAS and dragging it over to the graph.
