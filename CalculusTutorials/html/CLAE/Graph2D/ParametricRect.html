Parametric Equations (x, y) = (f(t), g(t))#

Description#

This type is for graphing the parametric equations

\[\begin{split}\begin{array}{rcl} x & = & f{\left(x \right)}\\ y & = & g{\left(x \right)}\end{array}\end{split}\]

The independent variable in the expressions can be x or t but not both. The variable y cannot be in the expressions either. All other variables are considered constants. In the CAS, the expressions can be input either as a list or as a \(2 \times 1\) matrix with the first entry the expression for the x coordinate and the second entry the expression for the y coordinate. For example, \(\left[ \sin{\left(x \right)}, \ \cos{\left(x \right)}\right]\) or

\[\begin{split}\left[\begin{array}{c}\sin{\left(x \right)}\\\cos{\left(x \right)}\end{array}\right]\end{split}\]

will give the parametric equations for

\[\begin{split}\begin{array}{rcl} x & = & \sin{\left(t \right)}\\ y & = & \cos{\left(t \right)}\end{array}\end{split}\]

Insert/Edit Dialog#

The Insert/Edit Dialog for parametric equations is pictured below.

The first inputs are the x and y coordinate expressions, then the lower and upper bounds on the parameter values, and finally the number of points and the line width options.

Options#

Parameter Minimum and Maximum#

Lower and upper bounds for the parameter can be any legitimate expression that evaluates to a real number. For example, 1.234, 3*pi, 1/E, etc. For example, if we graph,

Points#

This program was built with the goal of quick graphics, for this reason we do not use adaptive methods to graph curves (which give better images but are slower to plot) we simply take a number of evenly-spaced points over the domain of the graphics region. The number of points you select are the number of evaluations done over the graphing range. If you want more resolution to the plot, simply increase the number of points. As expected, the larger number of points used will slow down animations, zooming, and translations of the graphics view. The default value for most of the curves in this program is 500 points, which is good for most reasonably smooth curves.

Line Width#

The line width is simply the width of the line used for the plot. The default value for most of the curves in this program is 2, which is good for viewing the curve in the program interface and for exporting to images.
