View Options#

The options menu contains a few options for the look and feel of the editing area.

Increase Font Size#

Increases the editor font size.

Decrease Font Size#

Decreases the editor font size.

Reset Font Size#

Resets the editor font size to the default.

Toggle Bold#

Toggles the bold setting for the editor font.

Toggle Italic#

Toggles the italic setting for the editor font.

Font Size#

Opens a dialog box for the user to set the font size, 5-72.

Select Font#

Opens a dialog box for the user to select a new font family.
