Calculus & Linear Algebra Explorer User’s Guide#

Guide Navigation#

  • Introduction: Introduction, program layout, and general information on getting started with the Calculus and Linear Algebra Explorer (CLAE) program.

  • CAS: Guide to the use, syntax, and options for the Computer Algebra System segment of CLAE.

  • 2D Graphs: Information about the 2-D graphical interface. Contains descriptions of the objects that can be graphed, the graphical system layout and manipulation, importing and exporting, and animations.

  • 3D Graphs: Information about the 3-D graphical interface. Contains descriptions of the objects that can be graphed, the graphical system layout and manipulation, importing and exporting, and animations.

  • Spreadsheet: Information about the spreadsheet-like editing interface. Contains editing options, layout options, importing and exporting and calculations.

  • Text Editor: Information about the text editor interface. Contains editing options, data conversions, importing and exporting and pseudo-random number creation.

  • LaTeX Table Editor: Information about the LaTeX Table Editor interface. Contains editing options, LaTeX exporting environments and options, and other types of information exporting.
