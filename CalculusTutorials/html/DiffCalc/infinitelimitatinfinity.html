Infinite Limits at Infinity#

Discussion & Definitions#

Another examination of the end behavior of a function could be that the function increases or decreases without bound as x increases or decreases without bound. If this is the case we have infinite limits at infinity. More precisely,

Definition: Infinite Limits at Infinity

  • Let \(f(x)\) be a function defined at all values in an open interval \((c, \infty)\). If the values of the function \(f(x)\) increase without bound as the values of x increase without bound, then we say that the limit of f(x) as x approaches \(\infty\) is \(\infty\). That is, as x gets arbitrarily large, \(f(x)\) gets arbitrarily large. Symbolically we write,

\[\lim_{x \to \infty} f(x) = \infty\]

  • Let \(f(x)\) be a function defined at all values in an open interval \((c, \infty)\). If the values of the function \(f(x)\) decrease without bound as the values of x increase without bound, then we say that the limit of f(x) as x approaches \(\infty\) is \(-\infty\). That is, as x gets arbitrarily large, \(f(x)\) gets arbitrarily small (that is, large negatively). Symbolically we write,

\[\lim_{x \to \infty} f(x) = -\infty\]

  • Let \(f(x)\) be a function defined at all values in an open interval \((-\infty, c)\). If the values of the function \(f(x)\) increase without bound as the values of x decrease without bound, then we say that the limit of f(x) as x approaches \(-\infty\) is \(\infty\). That is, as x gets arbitrarily small, \(f(x)\) gets arbitrarily large. Symbolically we write,

\[\lim_{x \to -\infty} f(x) = \infty\]

  • Let \(f(x)\) be a function defined at all values in an open interval \((-\infty, c)\). If the values of the function \(f(x)\) decrease without bound as the values of x decrease without bound, then we say that the limit of f(x) as x approaches \(-\infty\) is \(-\infty\). That is, as x gets arbitrarily small, \(f(x)\) gets arbitrarily small (that is, large negatively). Symbolically we write,

\[\lim_{x \to -\infty} f(x) = -\infty\]

Just as with infinite limits these limits do not exist. We use the \(\pm\infty\) notation to portray more information about the function than simply DNE.

Example: \(f(x) = x^{3} - 3 x^{2} + x - 1\)#

GeoGebra#

To graph the function you can use the following syntax,

x^3-3 x^2 + x -1

From the graph it appears that

\[\lim_{x \to \infty} x^3-3 x^2 + x -1 = \infty \quad {\rm and } \quad \lim_{x \to -\infty} x^3-3 x^2 + x -1 = -\infty\]

We can also attack this algebraically. Select a new cell and type in limit select the limit option, put in f as your function and inf as the point (inf in GeoGebra is \(\infty\)) the result will be \(\infty\). Do the same in the next cell except use -inf as the limit point and again the result is \(-\infty\). This algebraically confirms our conclusions above.

CLAE#

To graph the function you can use the following syntax,

From the graph it appears that

\[\lim_{x \to \infty} x^3-3 x^2 + x -1 = \infty \quad {\rm and } \quad \lim_{x \to -\infty} x^3-3 x^2 + x -1 = -\infty\]

We can also attack this algebraically. Select the expression then select Calculus > Limit, variable is x and the limit point is oo, the result is \(\infty\). Now do the same but with a limit point of -oo, the result is \(-\infty\). This algebraically confirms our conclusions above.

Maxima#

To graph the function you can use the following syntax,

kill(all)

f(x):=x^3-3*x^2 + x -1

wxplot2d([f(x)], [x,-3,5], [y,-60,60])

From the graph it appears that

\[\lim_{x \to \infty} x^3-3 x^2 + x -1 = \infty \quad {\rm and } \quad \lim_{x \to -\infty} x^3-3 x^2 + x -1 = -\infty\]

We can also attack this algebraically by running the following two commands.

limit(f(x),x,inf)

limit(f(x),x,-inf)

This algebraically confirms our conclusions above.

Example: \(f(x) = \frac{x^{3} - 3 x^{2} + x - 1}{x^{2} + 1}\)#

GeoGebra#

To graph the function you can use the following syntax,

(x^3-3 x^2 + x -1)/(x^2+1)

From the graph it appears that

\[\lim_{x \to \infty} \frac{x^{3} - 3 x^{2} + x - 1}{x^{2} + 1} = \infty \quad {\rm and } \quad \lim_{x \to -\infty} \frac{x^{3} - 3 x^{2} + x - 1}{x^{2} + 1} = -\infty\]

Using the commands outlined in the first example, verify these algebraically. Note that with this function there is no horizontal asymptote but the curve does straighten out at the ends to what looks like a straight line. Since this function is a rational function we can do long division on it using the partial fractions function. In a new cell start typing PartialFractions, select the PartialFractions from the popup menu and input f as the function. The result will be,

\[x - 3 + \frac{2}{x^{2} + 1}\]

Plot \(x-3\) along with the original function and you see that the original function has this straight line as an asymptote. We would call this a slanted asymptote, tilted asymptote, or in general an asymptotic function.

Example: \(f(x) = x + \tan^{-1}{\left(x \right)} - 2\)#

The above method of partial fractions (really just long division) works well for rational expressions since it rewrites the function as \(f(x) = g(x) + h(x)\) where the function \(h(x)\) has a horizontal asymptote of \(y = 0\). So as x grows the value of \(h(x)\) gets very small and then the difference between \(f(x)\) and \(g(x)\) becomes negligible.

As you can guess, there are many more possibilities out there. For example, the function \(f(x) = x + \tan^{-1}{\left(x \right)} - 2\).
