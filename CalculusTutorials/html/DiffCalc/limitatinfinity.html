Limits at Infinity#

Discussion & Definitions#

The term limit at infinity is really asking the question, what is a function doing as the values of x get arbitrary large either positively or negatively. Another way to say this is what is the end behavior of the function, what is it doing when its graph goes off the screen in either direction. Remember that infinity is not a number so we cannot approach infinity in tht sense. When we say we are approaching infinity we really mean we are letting x get arbitrarily large. With this in mind we define our limits at infinity.

Definition: Limits at Infinity

  • Let \(f(x)\) be a function defined at all values in an open interval \((c, \infty)\). If the values of the function \(f(x)\) approach the real number L as the values of x increase without bound, then we say that the limit of f(x) as x approaches \(\infty\) is L. That is, as x gets arbitrarily large, \(f(x)\) gets closer to L. Symbolically we write,

\[\lim_{x \to \infty} f(x) = L\]

  • Let \(f(x)\) be a function defined at all values in an open interval \((-\infty, c)\). If the values of the function \(f(x)\) approach the real number L as the values of x decrease without bound, then we say that the limit of f(x) as x approaches \(-\infty\) is L. That is, as x gets arbitrarily small (that is, large negatively), \(f(x)\) gets closer to L. Symbolically we write,

\[\lim_{x \to -\infty} f(x) = L\]

For example, if we graph,

\[f(x) = \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1}\]

Looking at the graph at the two ends it does appear that the graph is leveling out to a single number \(L = 3\) off to the right. It also appears that the graph is leveling out to a single number \(L = 3\) off to the left. In our notation we would write,

\[\lim_{x \to \infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3 \quad {\rm and} \quad \lim_{x \to -\infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3\]

We can also attack this numerically, we simply let the x values get large and plug them into the function to see the corresponding outputs. From the values in the second column we would conclude that same as we did with the graph.

\[\begin{split}\begin{array}{l|l} x & \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} \\ \hline 1.0 & 1.33333333333333 \\ 10.0 & 2.87387387387387 \\ 100.0 & 2.98970398970399 \\ 1000.0 & 2.998997003999 \\ 10000.0 & 2.999899970004 \\ 100000.0 & 2.9999899997 \\ 1000000.0 & 2.999998999997 \\ 10000000.0 & 2.99999989999997 \\ 100000000.0 & 2.99999999 \\ 1000000000.0 & 2.999999999 \\ \end{array}\end{split}\]
\[\begin{split}\begin{array}{l|l} x & \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} \\ \hline -1.0 & 0 \\ -10.0 & 3.06593406593407 \\ -100.0 & 3.00969599030401 \\ -1000.0 & 3.000996995999 \\ -10000.0 & 3.000099969996 \\ -100000.0 & 3.0000099997 \\ -1000000.0 & 3.000000999997 \\ -10000000.0 & 3.00000009999997 \\ -100000000.0 & 3.00000001 \\ -1000000000.0 & 3.000000001 \\ \end{array}\end{split}\]

As another example, if we graph,

\[f(x) = 1 + \frac{\cos{\left(x \right)}}{x}\]

Looking at the graph at the two ends it does appear that the graph is leveling out to a single number \(L = 1\) off to the right. It also appears that the graph is leveling out to a single number \(L = 1\) off to the left. In our notation we would write,

\[\lim_{x \to \infty} 1 + \frac{\cos{\left(x \right)}}{x} = 1 \quad {\rm and} \quad \lim_{x \to -\infty} 1 + \frac{\cos{\left(x \right)}}{x} = 1\]

Definition: Horizontal Asymptote

If either

\[\lim_{x \to \infty} f(x) = L \quad {\rm or } \quad \lim_{x \to -\infty} f(x) = L\]

then we say that the line \(y = L\) is a horizontal asymptote of \(f\).

Going back to our previous examples. If we graph,

\[f(x) = \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1}\]

Along with the line \(y = 3\) we get,

Notice how the two lines get close to each other at the two ends. This is really what an asymptote means, a function that another function gets arbitrarily close to. Hence the line \(y = 3\) is the horizontal asymptote to our function.

If we graph,

\[f(x) = 1 + \frac{\cos{\left(x \right)}}{x}\]

Along with the line \(y = 1\) we get,

Hence the line \(y = 1\) is the horizontal asymptote to our function.

On the other hand, if we graph

\[f(x) = \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1}\]

It does not look like the function is leveling out at the ends, and it is not. So we would say that,

\[\lim_{x \to \infty} \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1} \quad DNE \quad {\rm and } \quad \lim_{x \to -\infty} \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1} \quad DNE\]

and consequently the function does not have a horizontal asymptote. In the next section we will investigate this more closely but from the graph it appears we can go a little further and say,

\[\lim_{x \to \infty} \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1} = \infty \quad {\rm and } \quad \lim_{x \to -\infty} \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1} = -\infty\]

Although this function does not have a horizontal asymptote, that does not mean that it does not heva an asymptote. If do long division on the function we get,

\[f(x) = x + 6 - \frac{7 x + 7}{x^{2} + x + 1}\]

If we remove the fraction at the end and plot \(y = x+6\) along with the original function we get,

Clearly these two curves are getting close to each other as x gets arbitrarily large. So the line \(y = x+6\) is an asymptote to our function. These are called slanted asymptotes or oblique asymptotes. This can be generalized further and we will in subsequent sections. Another visualization we can make about this slanted asymptote is if we graph the difference between the two functions,

\[\frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1} - (x+6)\]

we get,

Since this function has a horizontal asymptote at \(y = 0\) our original function has \(y = x+6\) as its slanted asymptote.

Example: \(f(x) = \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1}\)#

GeoGebra#

To graph the function you can use the following syntax,

(3 x^2+2 x-1)/(x^2+x+1)

Using the spreadsheet tool and methods from the previous sections we could produce the same numerical attack, we will not do that here. We can also attack this algebraically, GeoGebra can do infinite limits. Select a new cell and type in limit select the limit option, put in f as your function and inf as the point (inf in GeoGebra is \(\infty\)) the result will be 3. Do the same in the next cell except use -inf as the limit point and again the result is 3. This algebraically confirms our conclusions.

\[\lim_{x \to \infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3 \quad {\rm and} \quad \lim_{x \to -\infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3\]

CLAE#

To graph the function you can use the following syntax,

(3*x^2+2*x-1)/(x^2+x+1)

Using the spreadsheet tool and methods from the previous sections we could produce the same numerical attack, we will not do that here. We can also attack this algebraically, CLAE can do infinite limits. Select the function in the CAS, then select Calculus > Limit, variable is x and the limit point id oo (two small o’s represent \(\infty\) in CLAE). The direction does not need to be changed, the infinity overrides the direction selection. The result will be 3. Do the same again but this time use -oo for \(-\infty\) and again we get the result of 3. This algebraically confirms our conclusions.

\[\lim_{x \to \infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3 \quad {\rm and} \quad \lim_{x \to -\infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3\]

Maxima#

To graph the function you can use the following syntax,

kill(all)

f(x):=(3*x^2+2*x-1)/(x^2+x+1)

wxplot2d([f(x)], [x,-50,50], [y,-5,5])

For an algebraic approach run the following commands,

limit(f(x),x,inf)

and

limit(f(x),x,-inf)

or

limit(f(x),x,minf)

Each of these commands will return 3, algebraically confirms our conclusions.

\[\lim_{x \to \infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3 \quad {\rm and} \quad \lim_{x \to -\infty} \frac{3 x^{2} + 2 x - 1}{x^{2} + x + 1} = 3\]

Example: \(f(x) = \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1}\)#

GeoGebra#

To graph the function you can use the following syntax,

(x^3+7 x^2-1)/(x^2+x+1)

Use the algebraic methods in the first example to algebraically calculate the limits at infinity. All of our methods lead us to the conclusions that both infinite limits do not exist.

From the discussion above about this function we know that there is a slanted asymptote \(y = x+6\) to this function. We determined that by doing long division on the expression,

\[f(x) = \frac{x^{3} + 7 x^{2} - 1}{x^{2} + x + 1}\]

resulting in

\[f(x) = x + 6 - \frac{7 x + 7}{x^{2} + x + 1}\]

In GeoGebra we can do this calculation with the partial fractions function. Start typing in partialfractions select the PartialFractions from the popup menu, and input f as the function and you will get the above result. Note that the PartialFractions function does more than just long division but that is part of it. Now to graph the asymptote just type in x+6 into a new cell or select Duplicate Output from the cell’s menu and then remove the fractional portion in the new cell. Either way you will get.

Example: \(f(x) = \tan^{-1}(x)\)#

It is possible to have two different horizontal asymptotes for a function, one to the right and a different one to the left, \(f(x) = \tan^{-1}(x)\) is an example of such a function.

GeoGebra#

To graph the function you can use the following syntax,

atan(x)

From the graph it appears that there is a horizontal asymptote to the right at about 1.5 and one to the left at about \(-1.5\). If we use the limit command we get 1.5708 and -1.5708. The exact values here are \(\pi/2\) and \(-\pi/2\). GeoGebra will sometimes approximate values.

CLAE#

To graph the function you can use the following syntax,

atan(x)

From the graph it appears that there is a horizontal asymptote to the right at about 1.5 and one to the left at about \(-1.5\). If we use the limit command we get the exact values of \(\pi/2\) and \(-\pi/2\). We can graph these with the curve fairly easily, just put the two values in a list,

[-pi/2, pi/2]

Then drag them from the CAS to the graph, they should come in as horizontal lines,
