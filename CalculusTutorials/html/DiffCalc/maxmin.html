Extreme Values#

Discussion & Definitions#

Finding the largest and smallest values of a function is critical to solving optimization problems, such as, minimizing the materials in construction or maximizing profit.

We will start with absolute extremum.

Definition: Absolute Extremes

Let \(f\) be a function defined over an interval \(I\) and let \(c \in I\). We say \(f\) has an absolute maximum on \(I\) at \(c\) if \(f(c) \geq f(x)\) for all \(x \in I\). We say \(f\) has an absolute minimum on \(I\) at \(c\) if \(f(c) \leq f(x)\) for all \(x \in I\). If \(f\) has an absolute maximum on \(I\) at \(c\) or an absolute minimum on \(I\) at \(c\), we say \(f\) has an absolute extremum on \(I\) at \(c\).

A couple things to note before we proceed,

  • A quick note about the language here. If a function \(f\) has an absolute extremum over an interval \(I\) at \(c\), the absolute extremum is \(f(c)\). The real number \(c\) is a point in the domain at which the absolute extremum occurs.

  • Another thing to note is that an absolute extremum not only relies on the function but also the interval \(I\). So a function could have an absolute extremum on one interval but not on another.

  • There could be several places where the function attains an absolute maximum and/or its absolute minimum.

Let’s look at a few examples to get a feel for the possibilities.

Take the function, \(f(x) = \frac{x}{x^{2} + 1}\), graph is below.

  • On the interval \((-\infty, \infty)\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and an absolute minimum of \(-1/2\) at \(x = -1\).

  • On the interval \([-10, 10]\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and an absolute minimum of \(-1/2\) at \(x = -1\).

  • On the interval \([0, \infty)\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and an absolute minimum of \(0\) at \(x = 0\).

  • On the interval \((0, \infty)\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and no absolute minimum.

  • On the interval \([1, 5]\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and an absolute minimum of \(5/26\) at \(x = 5\).

  • On the interval \((1, 5]\), the function has no absolute maximum and an absolute minimum of \(5/26\) at \(x = 5\).

  • On the interval \((1, 5)\), the function has no absolute maximum and no absolute minimum.

Take the function, \(f(x) = x^3\), graph is below.

  • On the interval \((-\infty, \infty)\), the function has no absolute maximum and no absolute minimum.

  • On the interval \((0, \infty)\), the function has no absolute maximum and no absolute minimum.

  • On the interval \([0, \infty)\), the function has no absolute maximum and an absolute minimum of 0 at \(x = 0\).

Take the function, \(f(x) = \sin(x)\), graph is below.

  • On the interval \((-\infty, \infty)\), the function has an absolute maximum of 1 at \(\frac{\pi}{2} + 2\pi k\) and an absolute minimum of \(-1\) at \(\frac{3\pi}{2} + 2\pi k\). So there are an infinite number of places where the function attains its absolute maximum and its absolute minimum.

  • On the interval \([0, 4\pi]\), the function has an absolute maximum of 1 at \(\frac{\pi}{2}\) and \(\frac{5\pi}{2}\) and an absolute minimum of \(-1\) at \(\frac{3\pi}{2}\) and \(\frac{7\pi}{2}\).

With all of these possibilities it is nice to have some general results that guarantee the existence of an absolute maximum and minimum, the Extreme Value Theorem does just that.

Theorem: Extreme Value Theorem

If \(f\) is a continuous function over the closed, bounded interval \([a, b]\), then there is a point in \([a, b]\) at which \(f\) has an absolute maximum over \([a, b]\) and there is a point in \([a, b]\) at which \(f\) has an absolute minimum over \([a, b]\).

For example, with \(f(x) = \frac{x}{x^{2} + 1}\),

  • On the interval \([-10, 10]\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and an absolute minimum of \(-1/2\) at \(x = -1\).

  • On the interval \([1, 5]\), the function has an absolute maximum of \(1/2\) at \(x = 1\) and an absolute minimum of \(5/26\) at \(x = 5\).

Definition: Local Extremes

A function \(f\) has a local maximum at \(c\) if there exists an open interval \(I\) containing \(c\) such that \(I\) is contained in the domain of \(f\) and \(f(c) \geq f(x)\) for all \(x \in I\). A function \(f\) has a local minimum at \(c\) if there exists an open interval \(I\) containing \(c\) such that \(I\) is contained in the domain of \(f\) and \(f(c) \leq f(x)\) for all \(x \in I\). A function \(f\) has a local extremum at \(c\) if \(f\) has a local maximum at \(c\) or \(f\) has a local minimum at \(c\).

For example, take \(f(x) = x^3-x\), graph below,

On \((-\infty, \infty)\) the function has no absolute maximum and no absolute minimum. But it does have a local maximum of \(\frac{2 \sqrt{3}}{9}\) at \(x = - \frac{\sqrt{3}}{3}\) and a local minimum of \(-\frac{2 \sqrt{3}}{9}\) at \(x = \frac{\sqrt{3}}{3}\). Although we can simply see this, if we follow the definition to the letter we need to come up with a couple intervals. For the local maximum we can use \((-1, 0)\) (or \((-0.75, -0.25)\) or \((-0.6, -0.5)\)). For the local minimum we can use \((0, 1)\), amongst others.

As another example, \(f(x) = x^{5} - 3 x^{3} + x\), graph below,

This function has two local maximums and two local minimums. The local maximums are

\[\frac{\left(7 \sqrt{10} + 3 \sqrt{610}\right) \sqrt{\sqrt{61} + 9}}{250} \approx 1.57819155662093 \quad {\rm at} \quad x = - \sqrt{\frac{\sqrt{61}}{10} + \frac{9}{10}} \approx -1.29654346922526\]

and

\[\frac{\sqrt{9 - \sqrt{61}} \left(- 7 \sqrt{10} + 3 \sqrt{610}\right)}{250} \approx 0.226696737096979 \quad {\rm at} \quad x = \sqrt{\frac{9}{10} - \frac{\sqrt{61}}{10}} \approx 0.344927575600059\]

The local minimums are

\[\frac{\sqrt{9 - \sqrt{61}} \left(- 3 \sqrt{610} + 7 \sqrt{10}\right)}{250} \approx -0.226696737096979 \quad {\rm at} \quad x = - \sqrt{\frac{9}{10} - \frac{\sqrt{61}}{10}} \approx -0.344927575600059\]

and

\[\frac{\sqrt{\sqrt{61} + 9} \left(- 3 \sqrt{610} - 7 \sqrt{10}\right)}{250} \approx -1.57819155662093 \quad {\rm at} \quad x = \sqrt{\frac{\sqrt{61}}{10} + \frac{9}{10}} \approx 1.29654346922526\]

One more example, consider the following function,

It has a local maximum of about 1.4 around \(x = -0.6\), a local minimum of about 0.6 around \(x = 0.6\), and another local maximum of 3 at \(x = 2\).

It should be fairly clear that we are gearing up to find these local maximums and local minimums and of course it will relate to the derivative of the function. Take the above example, informally speaking, the local maximums and minimums occur at spots where the function “turns around”. There are essentially two ways that a continuous function can turn around, either smoothly were the derivative will exist or at a sharp corner where the derivative will not exist. If the derivative exists at the turnaround point the transition is smooth and if we think about tangent lines to the curve at that point the slope must be 0. Hence the derivative is 0. This observation gives us the following definition of a critical point or critical number as well as Fermat’s Theorem.

Definition: Critical Point

Let \(c\) be an interior point in the domain of \(f\). We say that \(c\) is a critical point or critical number of \(f\) if \(f'(c) = 0\) or \(f'(c)\) is undefined.

Theorem: Fermat’s Theorem

If \(f\) has a local extremum at \(c\) and \(f\) is differentiable at \(c\), then \(f'(c) = 0\).

As we have seen before with theorems, we need to be a little careful about how we read them. This is saying that if we have a local extreme then it must be at a critical number. It does not say that we must have an extreme at every critical number. Two very quick examples to illustrate this, first \(f(x) = x^3\), there is a critical value at \(x = 0\), since \(f'(x) = 3x^2\), and hence \(f'(0) = 0\), but there is no local maximum or minimum at \(x = 0\).

Another example is \(f(x) = \sqrt[3]{x}\), there is a critical value at \(x = 0\), since \(f'(x)\) does not exist there. \(f'(x) = \frac{1}{3 x^{\frac{2}{3}}}\) and hence \(f'(0)\) is undefined, but there is no local maximum or minimum at \(x = 0\).

Putting all of this together we have the following theorem about absolute extrema.

Theorem: Location of Absolute Extrema

Let \(f\) be a continuous function over a closed, bounded interval \(I\). The absolute maximum of \(f\) over \(I\) and the absolute minimum of \(f\) over \(I\) must occur at endpoints of \(I\) or at critical points of \(f\) in \(I\).

This theorem can be rewritten into an algorithm for finding absolute extrema.

Problem Solving: Locating Absolute Extrema over a Closed Interval

Consider a continuous function \(f\) defined over the closed interval \([a, b]\).

  1. Evaluate \(f\) at the endpoints \(x = a\) and \(x = b\).

  2. Find all critical points of \(f\) that are in the interval \((a, b)\) and evaluate \(f\) at those critical points.

  3. Compare all the functional values found in the previous steps, the largest is the absolute maximum of \(f\) and the smallest is the absolute minimum of \(f\).

Example: \(f(x) = x^3-x\)#

In this example we are going to find all the local maximums and minimums of the function \(f(x) = x^3-x\).

GeoGebra#

Input the function,

x^3-x

The quick way to get approximations to the local maximums and minimums, as well as roots and the y intercept, is to select Special Points from the objects menu.

You will note that the option ran three functions, Root, Extremum, and Intersect. You could have run these separately as well. One downside to this is that the displayed values are approximations. If you want exact values you can find the roots of the derivative, that is, solve \(f'(x) = 0\). To do so, in a new cell type in Solve, select Solve, the solve function asks for an equation you can input just f' or f'(x) or f'(x) = 0. The output will be the exact answers, if they can be calculated. You may also have noted that there is a Solutions function. This is the same as the Solve command but returns approximate values. Also, from the solve command the resulting cell has a button that will toggle between exact and approximate mode. This also shows you that the NSolve command could have been used in place of Solve which would start in approximate mode and toggle to exact mode.

Another note here is that the Solve command will return the solutions but will not graph them. The solutions are in list form, lets say they are l1. In a new cell input l1 and it will approximate the solutions as well as graph these as vertical lines. If you had use the Solutions command instead of the Solve command you would not need to do the approximation step.

One advantage ot using Solutions in stead of Solve is that you can find the corresponding y-values from the list result of Solutions but not from the list solutions from Solve. Had you used Solutions and say the list name was l2 then the y-values could be calculated using f(l2).

CLAE#

Input and graph the function,

x^3-x

To find the critical numbers first find the derivative, select the expression, select Calculus > Derivative, variable x, order 1. The result is \(3 x^{2} - 1\). Then solve the equation \(f'(x) = 3 x^{2} - 1 = 0\) by selecting Algebra > Solve, variable x and the result will be,

\[\left[ - \frac{\sqrt{3}}{3}, \ \frac{\sqrt{3}}{3}\right]\]

Click and drag these over to the graph and change the type to vertical lines.

To get the corresponding y-values there are two options,

Option 1: Select the original function, select Algebra > Evaluate, the variable is x, for the expression input R3[0]. This assumes that the solutions to the derivative equaling 0 are in the R3 result. What R3[0] will do is extract the first entry of the list stored in R3. Hence it will evaluate the function at the first solution. Then do the same thing again but this time use R3[1] to evaluate the second solution. Your results will be \(\frac{2 \sqrt{3}}{9}\) and \(-\frac{2 \sqrt{3}}{9}\) respectively.

Option 2: First extract all the solutions to their own output cells by selecting the solution list output, then select Edit > Extract from List > Extract All List Entries. Now evaluate each one by selecting the original function, then select Algebra > Evaluate, the variable is x, for the expression input R4 (assuming that the first entry went into R4). Then do the same but use R5 instead of R4. Your results will be \(\frac{2 \sqrt{3}}{9}\) and \(-\frac{2 \sqrt{3}}{9}\) respectively.

CLAE does have a feature to find both absolute and local maximums and minimums on finite intervals. Although this is quicker to a solution, the above methods help you understand the process a bit better. Nonetheless, once you understand the process and can do these calculations by hand, the machine can help you out with more difficult situations.

So input \(-1\) and 1 for the lower and upper bounds. For this exercise we want local extrema so select Local. We also want both the maximums and minimums, so select Both. For the next option we could have the program just output the y-values of the extremes or the points, we will want the points here so select Point. The options below this are to choose between exact numeric output or approximations. We will leave it as exact but if the program cannot produce exact solutions we will redo it with approximations. Click OK and we get,

\[\left[ \left[ - \frac{\sqrt{3}}{3}, \ \frac{2 \sqrt{3}}{9}\right], \ \left[ \frac{\sqrt{3}}{3}, \ - \frac{2 \sqrt{3}}{9}\right]\right]\]

kill(all);

f(rhs(sl[1]))

and the second can be done with,

f(rhs(sl[2]))

What is happening here is that the solution list was stored in sl from our solve command. Then sl[1] is the first entry, sl[2] is the second entry, and so on if the list was longer. Each entry is an equation and we want the right hand side of the equation, which is what the rhs function extracts. Once extracted we simple evaluate the function at that number. Your results should be the following, respectively.

\[\frac{2}{{{3}^{\frac{3}{2}}}}\mbox{} \quad {\rm and } \quad -\frac{2}{{{3}^{\frac{3}{2}}}}\mbox{}\]

Example: \(f(x) = \frac{x^{3} - 3 x^{2} + x - 1}{x^{2} - 4}\)#

For this example we will just find the critical numbers to \(f(x) = \frac{x^{3} - 3 x^{2} + x - 1}{x^{2} - 4}\).

GeoGebra#

Input the function,

(x^3 - 3 x^2 + x - 1)/(x^2 - 4)

Following the procedures from the first example, solving \(f'(x) = 0\), the Solve command reverts to approximation mode giving \(x = -4.3761\) and \(x = 0.16791\). In the previous examples we never needed to worry about the critical numbers from the derivative not existing. Since the derivative is,

\[\frac{x^{4} - 13 x^{2} + 26 x - 4}{x^{4} - 8 x^{2} + 16}\]

there is the possibility of division by 0. To extract the denominator go to a new cell, type in and select Denominator, then type in f'. It will extract, \(x^{4} - 8 x^{2} + 16\). Now use the Solve command and the results are \(-2\) and 2. These are where the derivative does not exist but we would not consider then critical numbers since the original function does not exist at these points either.
