:index:`Delete, Undo, and Redo`
===============================

Removing elements and the undo/redo options are fairly self explanatory but here are the details.

Delete
------

Delete will remove the currently selected item.

Delete All Error Entries
------------------------

This option will delete all error entries from the CAS workspace.

Undo
----

Undo simply undoes the last change to the CAS workspace.

Redo
----

Redo simply redoes the last undo to the CAS workspace.

.. note::

    If an edit is made after an undo operation the undo/redo history from the current CAS state on is removed and the new state is loaded at the end of the undo/redo history.
