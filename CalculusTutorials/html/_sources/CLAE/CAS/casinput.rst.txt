The Expression Input Bar
------------------------

The expression input bar is directly above the CAS workspace.

.. figure:: Images/CASPosition001.png
    :alt: Expression Input Bar and CAS Workspace

    Expression Input Bar and CAS Workspace


This is for inputting simple mathematical expressions and functions, piecewise expressions and matrices (and vectors) have their own input dialog boxes to make entering these structures easier and avoiding long syntax expressions. A detailed explanation of the expression syntax can be found in the quick :doc:`../CLAE/syntax` guide. We will go over these here as well. The icon to the right of the input bar is a clear button that will clear the bar.

The input bar is a drop-down box that holds the history of all the inputs.  So to load a previous input simply click the down-arrow and select the desired entry from the list.  This also comes in handy if you type in something and get a syntax error, which shows up in the CAS workspace as a red error entry, you can select the invalid input and edit it.  Another feature of this is that if you start to type something that has been entered before it will suggest an auto-completion.  If you do not want the auto-completion just hit the space key.
