File Options
============

Clear Graph
-----------

Clears the graph, graphics manager objects, and the slider list.  The graphs that are currently on the graph are stored in the undo/redo history and hence can be retrieved with an undo.

Open
-----------

Opens an Open dialog box for loading a 3-D graphics file into the graph area.  The loaded file overwrites the current graph, but the current graph is stored in the undo/redo history and hence can be retrieved with an undo.

Save as
-------

Opens a Save as dialog box for saving the current graph to a graph file.

Save Plot to Image File
-----------------------

Opens a Save as dialog box for saving the current graph as a png image file.

Copy Plot to Image
------------------

Copies the current graph to the system clipboard.

Print Plot
----------

This option will open the printer selection and properties dialog box allowing the user to select a printer as well as the options for the print job.  The print job will be of the plot image only.  If the image is too large for the print area it will be sized down to fit the available print area.

Print Preview Plot
------------------

This option will open the print preview dialog box.  The print job will be of the plot image only.  If the image is too large for the print area it will be sized down to fit the available print area.
