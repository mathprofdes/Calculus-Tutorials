View Options
============

Adjust Column Widths
--------------------

Adjusts the width of each column to fit the contents of the entries in the cells of the column.

Adjust Row Heights
------------------

Adjusts the height of each row to fit the contents of the entries in the cells of the row.

Adjust Row and Column Sizes
---------------------------

Adjusts both the width and height of each row and column to fit the contents of the entries in the table.

Reset Row and Column Sizes
--------------------------

Resets both the width and height of each row and column to initial cell size.

Increase Font Size
------------------

Increases the font size of the table font.

Decrease Font Size
------------------

Decreases the font size of the table font.

Reset Font Size
---------------

Resets the font size of the table font to its initial size.
