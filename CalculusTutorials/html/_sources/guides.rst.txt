The following are links to the user's guides for the software used in these tutorials.

.. container:: sphinx-features-three

    .. admonition:: `GeoGebra <CASGenIntro/GeoGebra.html>`_
        :class: sphinx-feature-three

        GeoGebra is a dynamic mathematics software for all levels of education that brings together geometry, algebra, spreadsheets, graphing, statistics and calculus in one engine.  For more information please see the `GeoGebra Website <http://geogebra.org/>`_.

    .. admonition:: `CLAE <CLAE/index.html>`_
        :class: sphinx-feature-three

        CLAE (Calculus and Linear Algebra Explorer) was developed for quick calculations and explorations in an introductory STEM mathematics classes, primarily single variable and multi-variable Calculus and Linear Algebra.

    .. admonition:: `wxMaxima <CASGenIntro/Maxima.html>`_
        :class: sphinx-feature-three

        wxMaxima is a document based interface for the computer algebra system Maxima. wxMaxima provides menus and dialogs for many common maxima commands, autocompletion, inline plots and simple animations. For more information please see the `wxMaxima Website <https://wxmaxima-developers.github.io/wxmaxima/>`_.
