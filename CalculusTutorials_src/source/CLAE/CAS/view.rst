:index:`View Options`
=====================

This submenu contains a few options for the CAS workspace.

Increase Font Size
------------------

This increases the font size of the CAS workspace.  This may come in handy if you are projecting this onto a presentation screen.

Decrease Font Size
------------------

This decreases the font size of the CAS workspace.

Reset Font Size
---------------

This resets the font size of the CAS workspace to the size given in the preferences.

Toggle Main Menu
----------------

This toggles the visibility of the main menu.  If you are working only with the graph tools and have hidden the CAS workspace you may wish to hide the menu and toolbar to increase the graphical footprint.  The shortcuts for toggling the menu and toolbar only work if the CAS workspace is visible.  Hence you may need to drag the CAS back into the program before you turn the menu and toolbar back on.

Toggle Main Toolbar
-------------------

This toggles the visibility of the main toolbar. If you are working only with the graph tools and have hidden the CAS workspace you may wish to hide the menu and toolbar to increase the graphical footprint.  The shortcuts for toggling the menu and toolbar only work if the CAS workspace is visible.  Hence you may need to drag the CAS back into the program before you turn the menu and toolbar back on.
