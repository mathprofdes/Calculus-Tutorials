Legend Options
==============

Toggle Legend
-------------

Toggles the legend on and off.   When there are a lot of graphics objects on the plot the legend can become long and interfere with viewing the graphs.

Increase Legend Font Size
-------------------------

Increases the font size used for the legend.

Decrease Legend Font Size
-------------------------

Decreases the font size used for the legend.

Reset Legend Font Size
----------------------

Resets the font size used for the legend.
