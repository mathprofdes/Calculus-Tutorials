Table Options
=============

The table options allow you to manipulate the data grid. Adding and deleting rows and columns, adjusting of cell sizes, and a few specialized operations.

Add Row Above
-------------

This adds a single row directly above the current selected position on the grid.

Add Row Below
-------------

This adds a single row directly below the current selected position on the grid.

Add Column Before
-----------------

This adds a single column directly before the current selected position on the grid.

Add Column After
----------------

This adds a single column directly after the current selected position on the grid.

.. note::

    There must be at least one cell currently selected on the grid or no row or column will be inserted. If multiple cells are selected the added row will be added directly above or below the selected block of cells. The same is true for added columns.


Delete Rows
-----------

This will delete all the rows that are in the currently selected block of cells.

Delete Columns
--------------

This will delete all the columns that are in the currently selected block of cells.

Delete Rows and Columns
-----------------------

This will delete all the rows and columns that are in the currently selected block of cells.

.. note::

    At least one cell must be selected for any rows or columns to be deleted.


Transpose Table
---------------

This options will transpose the table, convert the columns to rows and the rows to columns.

Trim Cells
----------

This removes the leading and trailing whitespace from each of the cells.

Fill Selected Cells with Text
-----------------------------

This will fill all the selected cells with text.  When this option is selected a dialog box will appear asking the user to input the text to insert into the selected cells.

Clear Selected Cells
--------------------

This clears the cells in the selected region.

Clear All
---------

The Clear All option will remove all data from the grid. Note that this does not clear the grid history. So if you mistakenly clear the grid an undo will bring the data back.
